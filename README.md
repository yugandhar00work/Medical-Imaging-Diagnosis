Application of AI in Medical Imaging Diagnosis
Project Overview:
Medical Imaging Diagnosis is a machine learning and web-based application developed to analyze medical images and assist in disease prediction.The system allows users
 to Upload medical images and receive prediction results.
Features:
Image preprocessing
Disease prediction using machine learning
Result visualization
User-friendly interface
Tech stack:
python
Django
NumPy
Pandas
Matplotlib
Machine Learning
Project Structure:
1-Code
2-Documents
3-Screenshots
4-Abstract
5-Base-Paper
Installation:
pip install -r requirements.txt
python manage.py runserver
How to Run:
Clone the repository
Install dependencies
Run the Django server
Open localhost in browser
